### Part 1: Arborescence (General Definition)

**there may be inaccuracies or simplifications in this article**

**1.1. Intuitive Representation:**
Imagine an ordinary undirected tree. Now assign a special role to one of its vertices — the **root**. Then orient all edges so that they point **outward from the root**, i.e., in the direction from the root to the leaves. The resulting structure is an arborescence.

### **Simple Example - Computer Folder Structure ⬇️**

- Root — `C:\` (or `/` on Linux)
- All folders are created via "New Folder"; there are no shortcuts or links.
- Each folder has **exactly one "parent"** (the folder it resides in).
- You can only go up to the parent. This is a classic **tree (arborescence)**.
